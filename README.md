# Spherical-Probe

UACJ Project (2015)

Construction of a dual rotor spherical drone, using 3D printers and development board chipKit uC32

---------------------------------------
## Details:

* C code for chipkit uC32
* C++/Python code for mechanical synthesis modeling and animations.
