# Spherical-Probe

UACJ Project (2015)

Construction of a dual rotor spherical drone, using 3D printers and development board chipKit uC32

----------------------------------------
## Chipkit uC32 Specs:

* PIC32MX340F512H processor
  * 512K Flash
  * 32K RAM
  * 80Mhz
